# ScreenshotUtil
Take a screen shot and recognize the text or scan the qrcode/barcode.
(only support Gnome now)

# Right Usage
Add a shortcut for it!  
Go to System Settings(`gnome-control-center`) -> Devices -> Keyboard -> Custom Shortcuts -> click `+`

# Requirements
+ Python 3.6+
 + Pillow
 + pyzbar
 + tesserocr
